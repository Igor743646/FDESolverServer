  • LeftBound - The variable L is the left boundary of the calculated grid

  • RightBound - The variable R is the right boundary of the calculated grid

  • MaxTime - The variable T is the max time boundary of the calculated grid

  • Alpha - The variable α is the fractional exponent of the degree of differential in the variable x

  • Gamma - The variable γ is the fractional exponent of the degree of differential in the variable t

  • SpaceStep - The variable h is the grid step by variable x

  • TimeStep - The variable τ is the grid step by variable x

  • Beta - The variable β is the coefficient of equilibrium between the left and right fractional derivatives

  • DiffusionCoefficient - The function D(x) is a function of the diffusion coefficient depending on the variable x

  • DemolitionCoefficient - The function V(x) is a linear demolition function depending on the variable x

  • ZeroTimeState - The function u(x, 0) is the state function at the zero point in time

  • SourceFunction - The function f(x, t) is the source function depending on the variables x and t

  • LeftBoundState - The function u(L, t) is the state function on the left boundary

  • RightBoundState - The function u(R, t) is the state function on the right boundary

  • BordersAvailable - Whether to include the flag when calculating the border or not

  • StochasticIterationCount - The number of iterations for the stochastic method if it is enabled

  • RealSolutionName - If the problem has a solution, then you can enter the formula of this function for visualization here in latex format

  • RealSolution - The formula of the real solution itself

  • SolveMethods - Choose solve methods: MGL, MRL, SGL, SRL. Methods are specified without spaces separated by a colon.

       MGL - Matrix Grünwald-Letnikov method

       MRL - Matrix Riemann-Liouville method

       SGL - Stochastic Grünwald-Letnikov method

       SRL - Stochastic Riemann-Liouville method
